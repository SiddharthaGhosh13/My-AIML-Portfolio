## 🚀 Run Locally

```bash
# 1. Clone the repository
git clone https://github.com/SiddharthaGhosh13/My-Portfolio.git

# 2. Navigate into the project
cd My-Portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## ✉️ EmailJS Setup (Contact Form)

The contact form uses [EmailJS](https://emailjs.com) — no backend required.

1. Create a free account at [emailjs.com](https://emailjs.com)
2. Connect your Gmail under **Email Services**
3. Create two templates — One notification (to You) + One auto-reply (to Sender)
4. Find the `// Update Credentials` comment in `src/components/Contact.jsx` and replace:

```js
const SERVICE_ID  = "your_service_id"
const TEMPLATE_ID = "your_template_id"
const PUBLIC_KEY  = "your_public_key"
```

This Portfolio is deployed on **Vercel** with Automatic Deployments on every push to `main`.

```bash
git commit -m "Update: Message"
git push
```
